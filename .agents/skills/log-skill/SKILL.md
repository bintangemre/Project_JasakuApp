## What I do

- Mencatat perkembangan skill pemrograman, framework, dan perkakas development (seperti Flutter, Laravel, Express.js, PostgreSQL).
- Menyimpan rangkuman fungsionalitas, arsitektur, atau pustaka yang dikuasai.
- Menyediakan referensi cepat yang siap pakai untuk peninjauan diri (self-review) atau portofolio.
- Tambahkan log yang sudah di kerjakan ke dalam file progress.txt

## When to use me

Gunakan log ini setiap kali selesai mempelajari teknologi baru, menyelesaikan fitur kompleks, atau berhasil mengimplementasikan suatu arsitektur sistem.
Lakukan pembaruan secara berkala agar riwayat peningkatan keahlian terdokumentasi dengan baik.

## How I do it

1. Read `progress.txt` untuk melihat log terakhir
2. Append entry baru di bagian bawah file
3. Group berdasarkan tanggal, gunakan bullet points
4. Format: aksi (`Modified`, `Created`, `Fixed`, `Deleted`) + nama file + deskripsi

**PENTING:** Selalu append, jangan overwrite file yang ada
